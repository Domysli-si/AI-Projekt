# Notebook Price Data Collector

Tento projekt slouží ke sběru reálných dat o cenách použitých notebooků z veřejně dostupných webových zdrojů.

Projekt je první fází většího softwarového projektu, jehož cílem je vytvoření modelu strojového učení pro predikci ceny notebooků.

## Funkce
- Automatický sběr dat o noteboocích
- Ukládání dat do CSV
- Opakovatelné spuštění na Windows i Linuxu

## Spuštění
### Windows
run_scraper.bat

### Linux
./run_scraper.sh

## Výstup
CSV soubor v adresáři `data/raw/`
